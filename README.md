1. Ajoutez 1 points pour chaque nombre pair du tableau.
2. Ajoutez 3 points pour chaque nombre impair du tableau, sauf pour le chiffre 5.

Exemples :

- Entrée: [1,2,3,4,5] -> Sortie: 13
- Entrée: [17,19,21]  -> Sortie: 9
